# MANIFESTS

Manifests are used to describe a drone. They form part of the ux_manifest object as the __hw_model__ attribute. There purpose is to indicate to the FCNT and RR Client, what actions, and observations are acceptable.

# Primary Attributes

Primary attributes are required.

| Attribute | Description         | Example  | 
| --------- | ------------------- | -------- |
| hwmodel   | model name of drone |
